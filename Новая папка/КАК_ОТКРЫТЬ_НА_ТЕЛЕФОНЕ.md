# Как открыть сайт на мобильном телефоне

## Способ 1: Через локальный сервер (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Узнайте IP адрес вашего компьютера

**В Windows:**
1. Откройте командную строку (Win + R, введите `cmd`, нажмите Enter)
2. Введите команду: `ipconfig`
3. Найдите строку "IPv4-адрес" - это ваш IP (например: 192.168.1.100)

### Шаг 2: Запустите веб-сервер

**Вариант A: Используя Python (если установлен)**
```
cd "c:\Users\Kristina Kalyavina\Desktop\Новая папка"
python -m http.server 8000
```

**Вариант B: Используя Node.js (если установлен)**
Установите http-server глобально:
```
npm install -g http-server
```
Затем запустите:
```
cd "c:\Users\Kristina Kalyavina\Desktop\Новая папка"
http-server -p 8000
```

### Шаг 3: Откройте на телефоне

1. **Убедитесь, что телефон подключен к той же Wi-Fi сети, что и компьютер**
2. Откройте браузер на телефоне (Chrome, Safari, Firefox)
3. Введите в адресной строке: `http://ВАШ_IP:8000`
   Например: `http://192.168.1.100:8000`

## Способ 2: Через инструменты разработчика браузера (БЫСТРЫЙ СПОСОБ)

### Для тестирования мобильной версии прямо на компьютере:

1. Откройте `index.html` в браузере Chrome или Edge
2. Нажмите **F12** (или **Ctrl+Shift+I**) - откроются инструменты разработчика
3. Нажмите кнопку **Toggle device toolbar** (или **Ctrl+Shift+M**)
4. Выберите устройство из списка (iPhone, Samsung Galaxy и т.д.)
5. Вы увидите, как сайт выглядит на мобильном устройстве

---

## Способ 3: Просто открыть файл (ПРОСТОЙ, но ограниченный)

1. Откройте файл `index.html` в браузере на компьютере
2. Скопируйте путь к файлу (например: `file:///C:/Users/Kristina%20Kalyavina/Desktop/Новая%20папка/index.html`)
3. Отправьте этот путь себе на телефон (например, через сообщения)
4. Откройте ссылку на телефоне

**⚠️ Примечание:** Этот способ может работать не со всеми функциями сайта, так как некоторые браузеры блокируют доступ к локальным файлам.

---

## Способ 4: Загрузить в облако

1. Загрузите все файлы (index.html, styles.css, script.js) на бесплатный хостинг:
   - GitHub Pages
   - Netlify Drop
   - Vercel
   - Firebase Hosting
2. Получите публичный URL
3. Откройте этот URL на любом устройстве в любой точке мира

---

## Устранение проблем

**Не открывается на телефоне?**
- Проверьте, что телефон и компьютер в одной Wi-Fi сети
- Убедитесь, что брандмауэр Windows не блокирует соединение
- Попробуйте другой порт (например, 3000 или 8080)

**Нужно открыть порт в брандмауэре:**
1. Откройте "Брандмауэр Защитника Windows"
2. "Дополнительные параметры" → "Правила для входящих подключений"
3. "Создать правило" → выберите "Порт" → TCP → укажите порт (8000)
